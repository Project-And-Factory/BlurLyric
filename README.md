# 使用说明
## 正常用户
首次安装： 进入文件夹，使用命令行输入
```
npm install
```
正常使用
```
npm run el:serve
```


## 服务器端口映射提示：

在main.js中，修改第四行config.onWebServer改为true即,
```
var config = {
  onWebServer: false
}
//改成
var config = {
  onWebServer: true
}

```
 然后映射 http://localhost:18776/ 即可
